# LeapMotionCtrl
## About
UnityでLeapMotionを使ったジェスチャー操作を検出するサンプルです。

## Environment
* Unity ver.5.4.0f3(64bit)
* Windows10(64bit)
* [LeapMotion] UnityCoreAssets ver.4.1.2
